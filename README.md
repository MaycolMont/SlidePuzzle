## Name Conventions

**Variables**
- Locals and classes: snake_case
- Private variables: _snake_case
- Constants: UPPER_CASE
- Functions: snake_case
- Scenes and scripts: CamelCase

Declaration Order

1. classname
2. signals
3. export variables
4. constants
5. private and public variables
6. internal functions
7. private functions
8. public functions
